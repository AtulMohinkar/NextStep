# 🚀 NextStep - Career Guidance Web App

NextStep is a modern career guidance web application built using React (Vite).  
It helps students explore different career paths, required skills, and structured learning roadmaps in a simple and interactive way.

---

## 🌟 Features

- 🎯 Explore different career options  
- 📚 Structured learning roadmaps  
- 💡 Skill suggestions for each career  
- ⚡ Fast and responsive UI (Vite + React)  
- 📱 Mobile friendly design  

---

## 🛠️ Tech Stack

- React.js  
- Vite  
- JavaScript (ES6+)  
- CSS3  
- React Router DOM
